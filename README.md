BusyPHP使用说明
===============

`BusyPHP` 框架基于 `ThinkPHP6.0` 进行的开发，所以 `ThinkPHP6.0` 中的所有内置方都可以继续使用，具体请参考 [官方手册](https://www.kancloud.cn/manual/thinkphp6_0/1037479) 

## 安装方法

```shell script
composer create-project busyphp/framework ./项目目录名称
```

## 数据库安装

安装完毕后依据控制台提示进行数据库安装

## 卸载安装器

整体安装成功后可以删除 busyphp/install

```shell script
composer remove busyphp/install
```
